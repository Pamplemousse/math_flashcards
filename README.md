# Maths flashcards

Formules et définitions utiles pour le [master CSI](http://mastercsi.labri.fr/), sous forme de cartes à utiliser avec le [système Leitner](https://en.wikipedia.org/wiki/Leitner_system).


#### Générer le pdf

```bash
pdflatex cards.tex
# ou
texi2pdf cards.tex
```


#### Documentation

```bash
texdoc flashcards
```
